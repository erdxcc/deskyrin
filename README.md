Deskyrin — возврат тары с выплатой в USDC и подтверждением на Solana

Проблема:
Программы переработки часто опираются на «баллы» и закрытые учёты: пользователю неочевидна реальная ценность действия, а брендам и операторам сложно доказать честность учёта возвратов. Нет простого связного пути: скан → доказательство возврата → расчёт в стейблкоине.

Решение:
Веб-приложение, где пользователь подключает кошелёк, сканирует QR на упаковке, получает верифицируемое доказательство (mint / on-chain) и после сдачи в точке партнёра получает USDC. Для экосистемы и жюри: прозрачность, трассируемость, неизменяемая история событий — не обещания, а проверяемые записи в блокчейне.

Почему Web3 / Solana:
USDC — понятная награда в деньгах, а не в абстрактных баллах.
Solana — быстрые и дешёвые транзакции для массовых сценариев «много сканов / много возвратов».
On-chain proof снижает споры о том, «был ли возврат» и «кто его зафиксировал».

Что уже есть (MVP / демо):
Лендинг и пользовательский флоу (Connect → Scan QR → Settle).
Бэкенд и интеграция с цепочкой: скан, идемпотентность, mint + recycle по цепочке.
Роли: пользователь, партнёры (публичный список), сценарии для киоска/автоматизации.

План на следующий этап:
Расширение сети партнёров и сценариев сдачи.
Улучшение UX для не-крипто аудитории (объяснения, онбординг).
Метрики ESG / отчётность для брендов на основе on-chain событий.
